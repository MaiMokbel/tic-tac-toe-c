### Tic-Tac-Toe in C

A feature-rich console implementation of the classic Tic-Tac-Toe (XO) game developed in C. The project includes multiple game modes, colored terminal output, and different computer difficulty levels to provide an engaging gameplay experience.

### Features

- Human vs Computer mode
- Two-player mode
- Three difficulty levels
  - Easy
  - Normal
  - Hard
- Colored console interface
- Win detection
  - Rows
  - Columns
  - Diagonals
- Draw detection
- Choose to play as X or O
- Replay option after each game


### Difficulty Levels

### Easy
The computer selects random valid positions.

### Normal
The computer prioritizes strategic positions and attempts to block or complete winning lines.

### Impossible
The computer always starts first and follows an advanced strategy designed to make winning extremely difficult.


### Technologies Used
- C
- Standard C Library
- ANSI Escape Codes for colored terminal output



### How to Play

1. Select the desired game mode.
2. Choose your symbol (X or O).
3. Enter the row and column for your move.
4. Continue until someone wins or the board is full.
